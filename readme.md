# Vefforritun 2, 2022. Verkefni 1: Gagnavinnsla

## Setja upp

- git clone https://github.com/emh33/vef2-verkefni1.git
- cd vef2-verkefni1
- npm install
- npm run build
- npm run dev
- þá er hægt að nálgast síðuna á slóð http://localhost:3000/
